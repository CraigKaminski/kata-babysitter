# kata-babysitter
To install dependencies run:
```
npm install
```
To run tests run:
```
npm test
```
To calculate nightly charge run:
```
node calculate-charge.js
```
Example:
```
node calculate-charge.js
start time (hour only): 8
bedtime (hour only): 10
end time (hour only): 1
total charge: $56
```
